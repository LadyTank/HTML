Les extensions d'images

.tiff
Image pour le print et la publication papier.

.jpg, .jpeg
Image compressée définitive, pour les photographies, sans transparence.

.gif
Image compressée définitive, pour le dessin et les images animées, avec transparence.

.bmp
Image non compressée, image bitmap.

.png
Image compressée; photographie et dessin, png 8 sans transparence, png 27 avec transparence.

.svg
Image vectorielle.

.ico
image pour meta donnée, l'image prend place dans le title de la page html.
